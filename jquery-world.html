1. JQUERY

    • JQUERY는 JS 언어를 간편하게 사용할 수 있도록 단순화시킨 오픈 소스 기반의 JS 라이브러리이다.
    • DOM과 이벤트에 관한 처리를 손쉽게 구현, Ajax 응용 프로그램 및 플러그인도 빠르게 개발할 수 있다.
    • 주요 웹 브라우저의 구버전을 포함한 대부분의 브라우저에서 지원된다.
    • 애니메이션 효과나 대화형 처리를 간단하게 적용해줌, 같은 동작을 하는 프로그램을 더 짧은 코드로 구현

  2. 기본

    • 웹페이지에서 제이쿼리를 사용하기 위해서는 제이쿼리 파일을 먼저 웹페이지에 로드해야한다.
      1. 제이쿼리 파일을 다운받아 로드하는 방법
      2. CDN(Content Delivery Network)을 이용하여 로드하는 방법
    • 제이쿼리 파일을 받아 서버에 저장하고, script태그를 웹페이지의 head내에 삽입
    • CDN이란 웹 사이트의 접속자가 서버에서 콘텐츠를 다운받아야 할 때, 자동으로 가장 가까운 서버에서 다운받도록 하는 기술
      이 기술을 이용하면 특정 서버에 트래픽이 집중되지 않고, 콘텐츠 전송 시간이 매우 빨라지는 장점
    • $(선택자).동작함수();
      달러($) 기호는 제이쿼리를 의미하고, 제이쿼리에 접근할 수 있게 해주는 식별자
      선택자를 이용하여 원하는 HTML 요소를 선택하고, 동작 함수를 정의하여 선택된 요소에 원하는 동작을 설정
    • $()함수
      $() 함수의 인수로는 HTML 태그 이름뿐만 아니라, CSS 선택자를 전달하여 특정 HTML 요소를 선택할 수 있습니다
      이러한 $() 함수를 통해 생성된 요소를 제이쿼리 객체(jQuery object)라고 합니다
      $() 함수에 전달되는 인수는 반드시 따옴표("")를 사용한 문자열 형태로 전달되어야 한다.
    • Document 객체의 ready()메소드
      자바스크립트에서는 Window 객체의 onload() 메소드를 이용하여 문서가 모두 로드된 뒤에 코드가 실행되도록 설정
      마찬가지로 제이쿼리에서는 Document 객체의 ready() 메소드를 이용하여 같은 결과를 보장

  3. 요소 선택

    • 제이쿼리를 사용하면 손쉽게 HTML 요소를 선택하여, 선택된 요소에 특정 동작을 설정할 수 있다.
    • 태그 이름을 사용하여 같은 태그 이름을 가지는 HTML요소를 모두 선택할 수 있다.(getElementsByTagName()메소드와 같은 동작)
      $("span").css("fontSize", "28px");
    • 아이디(id)를 사용하여 특정 HTML 요소를 선택할 수도 있다.(getElementById())
      $("#jq").css("border", "2px solid orange");
    • 클래스(class)를 사용하여 같은 클래스에 속하는 HTML 요소를 모두 선택할 수 있다.(getElementsByClassName())
      $(".jq").css("backgroundColor", "lightgray");
    • 속성(attribute)을 사용하여 속성이 조건에 맞는 특정 HTML 요소를 선택할 수 있다.
      $("img[alt='flower']").attr("src", "/examples/images/img_monalisa.png");
    • 제이쿼리에서는 선택한 요소들을 변수에 저장하여 사용할 수 있다.
    • 또한, 선택한 요소 중에서 더욱 세분화된 선택을 하기 위한 필터링을 진행할 수 있다.
      :eq(n), :gt(n), :lt(n), :even...
    • 제이쿼리에서는 입력 양식(input)에 관련된 특정 요소를 손쉽게 선택할 수 있다.
      :button, :checkbox, :file, :image ....
    • getter메소드, setter메소드
      getter메소드는 선택된 요소에 접근하여 그 값을 읽어오기 위한 메소드, 아무런 인수를 전달하지 않고 호출
      setter메소드는 선택된 요소에 접근하여 그 값을 설정하는 메소드, 대입하고자 하는 값을 인수로 전달하여 호출
    • 메소드 체이닝(method chaining) : setter 메소드는 선택된 '모든 요소'에 인수로 전달된 값을 설정,
      그리고 선택된 모든 요소에 접근할 수 있는 또 다른 제이쿼리 객체를 반환
    • .html() / .text() / .width() / .height()메소드
    • .attr()메소드 : 선택한 요소의 특정 속성값을 반환하거나 설정하기 위해 사용
    • .position() : 선택한 요소 중에서 첫 번째 요소에 대해 특정 위치에 존재하는 객체를 반환(getter 메소드)
      .val() : form요소의 값을 반환하거나 설정함.

    5.         var currentIndex = 0; //슬라이드 인덱스 지정
        function moveSlide(index){  
            $("").children('li').css({'opacity':0});    //.children() : 어떤 요소의 자식 요소를 선택하는 메서드
            $("").children('li').stop();    // .stop() : 현재 동작하고 있는 애니메이션 즉시 중단
            $("").children('li').eq(index).stop().animate({'opacity':1}, 2000);
            currentIndex = index;}

        var SkillSize = $("").children('li').size();
        function prev(){
            if (currentIndex == 0){
                currentIndex = SkillSize - 1;
                moveSlide(currentIndex);
            }else{
                moveSlide(currentIndex - 1);}}
    
        function next(){
            if (currentIndex == SkillSize -1){
                currentIndex = 0;
                moveSlide(currentIndex);
            }else{
                moveSlide(currentIndex + 1);}}

        function SkillSlide(){ //슬라이드 자동 전환
            timer = setInterval(NextSlide, 3800)} 
            //setTimeout 일정 시간이 지난 후에 함수를 실행 
            //setInterval 일정 시간 간격을 두고 함수를 실행

        function Stop(){
            clearInterval(timer)}
    
        function NextSlide(){
            currentIndex++;
            if(currentIndex >= SkillSize){
                currentIndex = 0
            }
            moveSlide(currentIndex)}

        Skill.on('mouseenter', SkillSlide);
        Skill.on('mouseleave', Stop);

    6.     //Responsive menu
    $(window).resize(function(){
        var width = $(window).width();
        if (width < 1024){
            $("#menu").css({'display':'inline-block'});
            $(".menubar").children('a').css({'display':'none'});
        }else{
            $("#menu").css({'display':'none'});
            $(".menubar").children('a').css({'display':'inline-block'});
        }})
    $(window).trigger("resize");
    //.on()은 이벤트가 발생할 때 바로 가기 메서드 중 하나 또는 함께 연결된 모든 이벤트 핸들러가 트리거됩니다. 
    그러나 .trigger()메서드를 사용하여 수동으로 발사할 수 있습니다

    $("#menu").click(function(){
        $(this).nextAll("a").slideUp();
        if ($(this).nextAll("a").is(':hidden')){    //":hidden" : 숨겨진 모든 요소를 선택 //.is()선택한 요소에서 주어진 선택자가 있는지 판별
            $(this).nextAll("a").slideDown(1000);   //요소를 아래쪽으로 나타나게 하는 메서드
            $(".menubar").children('a').css({'display':'block'});
        }else{
            $(this).nextAll("a").slideUp();}}) //요소를 위쪽으로 사라지게 하는 메서드

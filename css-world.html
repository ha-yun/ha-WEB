Css Tag

  1. footer tag - p tag를 사용한 것과 화면상 다를게 없다.
  2. text-align: center; - 중앙으로 정렬
  3. float: left,right; - 붕붕떠다니면서 정렬
  4. overflow: hidden; - float를 묶어준다.
  5. background-color: black; - 배경색깔
  6. color: white; - 글자색깔
  7. font-size: 30px; - 글자 크기
  8. font-weight: lighter, bold,,; - 글자 무게
  9. font-style: italic; - 글자 스타일(글자 기울이기)
  10. line-height: 10px; - 줄간 간격 늘리기
  11. border: 두께 방식 색깔; - 상자그리기
  12. border-bottom: border-top: border-right: border-left: - 상하좌우에 선이 생김
  13. width: -상자 넓이조절
  14. height: -상자 높이 조절
  15. margin-left,margin-right: -상자위치 조절
  16. padding: 상자 크기와 글자 사이 거리 조절
  17. box-shadow: 가로축 세로축 그림자흐림정도 그림자퍼짐도 그림자의색(rgba)(r,g,b,투명도) - 상자에 그림자 넣는 코드

애니메이션

  • animation 속성은 특정 지점에서 스타일을 바꾸면서 애니메이션을 만드는데, 이렇게 애니메이션 중간에 스타일이 바뀌는 지점을 키프레임keyframe이라고 한다.
  • 키프레임은 @keyframes속성으로 정의하고, animation 속성과 그 하위 속성을 이용해서 애니메이션의 실행 시간이나 반복 여부 등을 지정한다.

Position

  • static속성은 대부분의 html태그들의 기본 속성, 정적인 위치를 갖는다.
  • fixed속성은 위치를 설정하지 않으면 (x=0,y=0)으로 설정한 것처럼 보여지게 된다.
  • sticky 속성은 static과 fixed속성의 특징을 모두 가지고 있는 속성이다.

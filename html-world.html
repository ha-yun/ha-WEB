HTML 구조

  • HTMl문서는 일반 문서와 달리 정해진 형식에 맞추어 내용을 입력해야 한다.
  • 웹 문서는 보통 !DOCTYPE html로 시작해 html, head, body라는 3개의 영역으로 구성되어 있다.
    1. !DOCTYPE html : 현재 문서가 HTML5언어로 작성한 웹 문서라는 뜻이다.
    2. html : 웹 문서의 시작과 끝을 나타내는 태그이다, 웹 브라우저가 html태그를 만나면 /html까지 소스를 읽어 화면에 표시한다.
    3. head : 웹 브라우저가 웹 문서를 해석하는데 필요한 정보를 입력하는 부분이다.
    4. body : 실제로 웹 브라우저 화면에 나타나는 내용
  • html태그에서 lang 속성으로 문서에서 사용할 언어를 지정할 수 있다, 검색 사이트에서 특정 언어로 제한해 검색할 때 필요,
    화면 낭독기에서 웹 문서를 소리 내어 읽어줄 때 다르게 할 수 있다.
  • head태그에는 meta, title 태그를 사용
    • meta는 데이터에 관한 데이터
    • meta charset='UTF-8' : 웹 서버는 영어가 기본이므로 화면에 한글로 된 내용을 표시할 때는
      UTF-8이라는 문자 세트를 사용한다고 웹 브라우저에 알려줘야 한다.
    • title은 웹 문서의 제목을 나타낸다.

  • 시맨틱 태그

    • HTML의 태그는 그 이름만 봐도 의미를 알 수 있어 시맨틱(semantic, 의미론적인)태그라고 한다.
    • 시맨틱 태그를 사용하면 웹 브라우저가 HTML의 소스 코드만 보고도 제목과 본문 메뉴를 쉽게 알 수 있다.
      또한, 문서 구조가 정확히 나눠지므로 PC나 모바일의 웹 브라우저와 여러 스마트 기기의 다양한 화면에서 웹 문서를 표현하기 쉽다.
      마지막으로, 인터넷에서 웹 사이트를 검색할 대 필요한 내용을 정확히 찾을 수 있다.
    • 헤더 영역을 나타내는 header, 내비게이션 영역을 나타내는 nav, 핵심 콘텐츠를 담는 main, 독립적인 콘텐츠를 담는 article,
      콘텐츠 영역을 나타내는 section,사이드 바 영역을 나타내는 aside, 푸터영역을 나타내는 footer, 여러 소스를 묶는 div
  • html은 문서의 요소요소를 구분, css는 화면상에서 적절한 곳에 있도록 구분
  • 같은 파일에 두 언어를 사용하는 것은 읽는데 피로감,, 따로 작성하도록 한다!

HTML 태그

  1. 텍스트 입력하기
    • hn : 제목을 나타내는 태그(n은 1~6)
    • p : 텍스트 단락을 만드는 태그
    • br : 줄을 바꾸는 태그
    • blockquote : 인용할 때 쓰는 태그
    • strong, b : 텍스트를 굵게 표시하는 태그
    • em, i : 기울인 텍스트를 입력해주는 태그
    • abbr, cite, code, small, sub, sup, s, u, ins, del
  2. 목록 만들기
    • ol, li : 순서 있는 목록을 만드는 태그
      ol 태그는 type속성을 사용하여 영문자나 로마 숫자 등으로 순서를 나타낼 수 있다.
      start 속성을 사용하면 시작 번호를 바꿀 수 있다.
    • ul, li : 순서 없는 목록을 만드는 태그
    • dl, dt, dd : 설명 목록을 만드는 태그(description list)
  3. 표 만들기
    • table, caption : 표를 만드는 태그(caption은 생략가능)
    • tr : 행을 만듬
    • td, th : 셀을 만드는 태그
    • thead, tbody, tfoot : 표의 구조를 지정하는 태그
    • rowspan, colspan : 행이나 열을 합치는 속성
    • col, colgroup : 열을 묶어주는 태그
  4. 이미지 삽입하기
    • img srg="이미지 파일 경로" alt="대체용 텍스트"
    • 이미지 크기를 조절하는 width, height 속성
  5. 오디오나 비디오 삽입하기
    • object, embed : 다양한 멀티미디어 파일을 삽입할 때 쓰는 태그
    • audio, video : 오디오나 비디오 파일을 삽입하는 태그
  6. 하이퍼링크 삽입하기
    • a href="링크할 주소"
    • 클릭하면 새로운 탭에서 사이트가 열리도록 target 속성을 _blank로 지정한다.

FORM

  1. 폼 삽입하기
    • form [속성="속성값"]
    • method : 사용자가 입력한 내용을 서버쪽 프로그램으로 어떻게 넘겨줄 것인지 지정(get, post)
      get - 데이터를 256~4,096byte까지만 서버로 넘길 수 있다, 주소 표시줄에 사용자가 입력한 내용이 그대로 드러나는 단점
      post - 입력한 내용의 길이에 제한받지 않고 사용자가 입력한 내용도 드러나지 않는다.
    • name : 자바스크립트로 폼을 제어할 때 사용할 폼의 이름을 지정
    • action : form태그 안의 내용을 처리해 줄 서버 프로그램을 지정
    • target : action 속성에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에서 열도록 한다.
    • autocomplete : 자동 완성 기능을 타나내는 속성
    • fieldset, legend : 폼 요소를 그룹으로 묶는 태그
    • label : 폼 요소에 레이블을 붙이는 태그
      - label 태그 안에 폼 요소를 넣는다.
      - label태그와 폼 요소를 따로 사용하고 label의 for속성과 폼 요소의 id속성을 이용해 서로 연결한다.
  2. 사용자 입력을 위한 input태그
    3. input 태그의 type 속성
    • text : 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 박스를 넣는다.
    • search : 검색할 때 입력하는 필드를 넣는다.
    • password,email, url, tel, date, month, week, time, datetime, datetime-local
    • checkbox : 주어진 여러 항목에서 2개 이상 선택할 수 있는 체크 박스를 넣는다.
      radio : 주어진 여러 항목에서 1개만 선택할 수 있는 라디오 버튼을 넣는다.
    • number : 숫자를 조절할 수 잇는 스핀 박스를 넣는다.
      range: 숫자를 조절할 수 있는 슬라이드 막대를 넣는다.
    • submit, reset, button, image : submit 버튼 대신 사용할 이미지를 넣는다.
    • file : 파일을 첨부할 수 있는 버튼을 넣는다.
      hidden : 사용자에게는 보이지 않지만 서버로 넘겨주는 값이 있는 필드를 만든다.

    • (text, password)에서 사용하는 주요 속성
      size : 화면에 몇 글자가 보이도록 할 것인지 지정
      value : 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 보여주는 내용, 지정하지 않으면 빈 텍스트 필드 표시
      maxlength : 입력할 수 있는 최대 문자 수 지정
    • (checkbox, radio)에서 사용하는 주요 속성
      value : 선택한 체크 박스나 라디오 버튼을 서버에 알려줄 때 넘겨줄 값 지정
      checked : 여러 항목 중에서 기본으로 선택해 놓고 싶은 항목에 사용
    • (number, range, 날짜나 시간)에서 사용할 수 있는 속성
      min : 필드에 입력할 수 있는 최솟값을 지정, 기본 최솟값은 0
      max : 필드에 입력할 수 있는 최댓값을 지정, 기본 최댓값은 100
      step : 숫자 간격을 지정, 기본값은 1
      value : 필드에 표시할 초깃값

    • autofocus : 자동으로 입력 커서를 갖다 놓는 속성
    • placeholder : 힌트를 표시해 주는 속성
    • readonly : 읽기 전용 필드를 만들어 주는 속성
    • required : 필수 입력 필드를 지정하는 속성
  3. 폼에서 사용하는 다양한 태그
    • textarea : 여러줄을 입력하는 텍스트 영역 태그
      cols - 너비 크기를 지정하는 속성
      rows - 세로길이를 줄 단위로 지정, 지정한 숫자보다 줄 개수가 많아지면 스크롤 막대가 생긴다.
    • select, option : 드롭다운 목록을 만들어 주는 태그
      select 태그로 드롭다운 목록의 시작과 끝을 표시하고, 그 안에 option 태그를 사용해 원하는 항목을 추가
      option 태그에는 value 속성을 이용해 서버로 넘겨주는 값을 지정
    • select태그를 사용해 만든 드롭다운 목록은 기본적으로 옵션이 하나만 표시되는데,
      size속성이나 multiple속성을 이용하면 드롭다운 목록의 크기나 선택할 항목의 개수를 조절할 수 있다.
    • option태그는 항목마다 value 속성을 사용해서 서버로 넘겨줄 값을 지정,
      기본적으로 목록에 보이도록 selected속성을 추가한다.
    • datalist, option : 데이터 목록을 만들어주는 태그
      datalist태그를 이용해 데이터 목록의 시작과 끝을 표시하고 그 사이에 option 태그를 사용해 각 데이터의 옵션을 표시한다.
      그리고 데이터 목록을 사용할 텍스트 필드에서 어떤 데이터 목록을 연결할지 id값을 지정한다.(datalist와 input text의 id연결)
    • button : 버튼을 만들어주는 태그
      submit - 폼을 서버로 전송, input type="submit"과 같은 기능
      reset - 폼에 입력한 내용을 초기화 input type="reset"과 같은 기능
      button - 버튼 형태만 만들뿐 자체 기능은 없다 input type="button"과 같은 기능
